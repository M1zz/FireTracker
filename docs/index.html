퇴사각

자산 · 순자산 · FIRE(조기은퇴) 목표를 한눈에 추적하는 앱

소개

퇴사각은 보유한 자산과 부채를 등록해 순자산과 총자산을 계산하고, 매달 기록을 쌓아 변화 추이와 FIRE(경제적 자유) 목표 달성률을 보여주는 개인 재무 추적 앱입니다. 모든 데이터는 사용자 기기에만 저장되며 별도의 회원가입이 없습니다.

주요 기능

  • 📊주식·코인·부동산·현금·연금·부채 등 카테고리별 자산 관리와 종목별 세부 기록
  • 📈월별 자산·부채·순자산 추이 그래프와 지난 기록 대비 변화
  • 🎯목표 지출·인출률 기반 FIRE 달성률과 올해 말 예측
  • 📡티커·종목코드를 넣으면 미국 주식·국내 주식·암호화폐·부동산 시세 자동 조회
  • 🖼️증권사 보유종목 스크린샷을 기기 내부에서 인식해 한 번에 추가

자주 묻는 질문

내 자산 데이터는 어디에 저장되나요?

모두 사용자의 기기 안(Apple SwiftData)에만 저장됩니다. 서버로 업로드되지 않으며 개발자도 볼 수 없습니다.

시세가 자동으로 안 불러와져요.

미국 주식·암호화폐·환율은 키 없이 동작합니다. 국내 주식·부동산 시세는 설정 탭에서 해당 API 키(한국투자증권·공공데이터포털)를 입력해야 합니다. 네트워크 연결과 올바른 티커/종목코드도 확인해주세요.

데이터를 삭제하려면?

앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.

총자산과 순자산의 차이가 뭔가요?

앱에서 ‘순자산’은 보유 자산의 합, ‘총자산’은 거기서 부채를 뺀 값(자산 − 부채)입니다.
