개인정보처리방침

앱: 퇴사각 · 시행일: 2026년 6월 12일

요약: 퇴사각은 회원가입이 없고, 사용자의 재무 데이터를 서버로 수집·저장하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 수집·저장하는 정보

사용자가 입력하는 자산·부채·수입·지출·목표 등 모든 재무 데이터는 전적으로 사용자의 기기 내부 (Apple SwiftData)에만 저장됩니다. 개발자를 포함한 제3자는 이 데이터에 접근할 수 없으며, 별도의 계정 생성이나 로그인이 없습니다. 앱은 이름·이메일·전화번호 등 개인 식별정보를 수집하지 않습니다.

2. 시세 자동 조회 시 외부로 전송되는 정보

‘시세 자동’ 기능을 사용할 때, 사용자가 입력한 조회 식별자(종목코드·티커·코인 심볼·법정동코드)가 현재가를 받아오기 위해 아래 외부 서비스로 전송됩니다.

  • Finnhub — 미국 주식 시세
  • 한국투자증권(KIS) Open API — 국내 주식 시세
  • 업비트(Upbit) — 암호화폐 시세
  • 공공데이터포털(국토교통부) — 부동산 실거래가
  • 환율 제공처 — 통화 환산

이때 전송되는 것은 조회 대상 식별자뿐이며, 보유 수량·평가액·자산 총액 등 개인 재무정보나 신원정보는 전송되지 않습니다. 각 서비스로 전송된 요청의 처리에는 해당 서비스의 개인정보 방침이 적용됩니다.

3. 스크린샷 가져오기

증권사 보유종목 스크린샷으로 자산을 추가하는 기능은 Apple의 온디바이스 Vision으로 기기 내부에서만 이미지를 인식하며, 이미지나 인식 결과를 외부로 전송하거나 저장하지 않습니다.

4. 분석·광고·추적

앱은 분석 도구, 광고, 사용자 추적(트래킹)을 사용하지 않으며, 사용 데이터를 수집하지 않습니다.

5. 데이터 보관 및 삭제

모든 데이터는 기기에 저장되므로, 앱을 삭제하면 데이터도 함께 삭제됩니다. 기기에서 iCloud 백업을 사용하는 경우 해당 백업은 Apple의 기기 백업 정책을 따릅니다.

6. 아동의 개인정보

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 방침 변경

본 방침이 변경되면 이 페이지에 갱신하여 게시합니다.
